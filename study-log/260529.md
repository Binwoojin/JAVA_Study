### 오늘 배운 핵심 개념

#### 1. Method Overloading
동일한 메서드 이름을 사용하면서 매개변수의 개수 또는 자료형을 다르게 정의하는 방법

##### 특징
<ul>
  <li>메서드 이름은 동일</li>
  <li>매개변수 타입 또는 개수가 달라야 함</li>
  <li>반환형만 다른 것은 오버로딩 불가</li>
</ul>

```java
plus(int a, int b);
plus(int a, float b, int c);
plus(int a, double b, int c);
```

---

#### 2. 매개변수 전달 방식

##### Call By Value
값을 복사하여 전달

```java
sum(100, 200);
```

##### 특징
<ul>
  <li>원본 값 변경 불가</li>
  <li>기본 자료형(int, double, boolean 등)</li>
</ul>

---

##### Call By Reference
참조값(주소값)을 전달

```java
findMax(m);
```

##### 특징
<ul>
  <li>배열, 객체 전달 시 사용</li>
  <li>원본 데이터 변경 가능</li>
</ul>

---

#### 3. 배열 활용
다양한 배열 관련 메서드 구현

##### 최대값 찾기

'''java
findMax(int[] m)
```
