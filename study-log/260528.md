# 26.05.28 Java Study

오늘은 배열(Array)과 2차원 배열을 중심으로 다양한 응용 문제를 풀며 배열 활용 방법을 학습

---

## 오늘 학습한 핵심 개념

### 1. 배열(Array)

같은 자료형의 여러 데이터를 하나의 변수명으로 관리하는 자료 구조

```java
int[] arr = new int[10];
```

#### 학습 내용
<ul>
  <li>배열 선언 및 생성</li>
  <li>배열 값 저장</li>
  <li>배열 값 출력</li>
  <li>반복문과 배열 함께 사용</li>
  <li>index를 활용한 데이터 접근</li>
</ul>

---

### 2. 반복문과 배열 활용

배열은 반복문과 함께 사용하는 경우가 많다.

```java
for(int i = 0; i < arr.length; i++)
```

#### 학습 내용
<ul>
  <li>배열 전체 출력</li>
  <li>특정 조건의 값만 출력</li>
  <li>배열 값 탐색</li>
  <li>배열 값 변경</li>
  <li>배열 데이터 비교</li>
</ul>

---

### 3. 조건문을 활용한 배열 데이터 처리

조건문을 사용하여 특정 조건의 값만 처리하였다.

#### 활용 문제
<ul>
  <li>3의 배수 출력</li>
  <li>5의 배수 출력</li>
  <li>3과 5의 공배수 처리</li>
  <li>특정 값의 index 찾기</li>
  <li>배열 값 비교</li>
</ul>

#### 핵심 개념

```java
if(arr[i] % 3 == 0)
```

```java
if(arr[i] == 50
```

---

### 4. 2차원 배열(Array[][])

행(row)과 열(column) 구조를 가지는 배열

```java
int [][] arr = new int [5][5];
```

#### 학습 내용
<ul>
  <li>2차원 배열 선언</li>
  <li>행과 열 반복 출력</li>
  <li>가로합 계산</li>
  <li>세로합 계산</li>
  <li>배열 값 채우기</li>
</ul>

#### 핵심 개념

```java
arr[i][j]
```

- i : 행(row)
- j : 열(column)

---

### 5. 배열 값 검색 (Search)

배열 안에서 특정 값을 찾는 방법을 학습하였다.

#### 활용 문제
<ul>
  <li>값이 50인 위치(index) 찾기</li>
</ul>

#### 핵심 개념

```java
if(arr[i] == 50)
```

---

### 6. 배열 값 변경

특정 조건에 따라 배열 값을 변경하는 방법을 학습하였다.

#### 활용 문제
<ul>
  <li>index가 3의 배수 -> 3333</li>
  <li>index가 5의 배수 -> 5555</li>
  <li>3과 5의 공배수 -> 3535</li>
</ul>

#### 핵심 개념

조건문의 순서가 중요하다.

```java
if(i % 3 == 0 && i % 5 == 0)
```

공배수 조건을 먼저 검사해야 한다.

---

### 종합문제 - 숫자 야구 게임

배열, 반복문, 조건문을 종합적으로 활용한 응용문제

#### 구현 내용
<ul>
  <li>랜덤 숫자 생성</li>
  <li>중복 제거</li>
  <li>사용자 입력 배열 저장</li>
  <li>Strike / Ball 판정</li>
  <li>게임 반복</li>
  <li>정답 시 종료</li>
</ul>

---

### 숫자 야구 개임 핵심 개념

#### Strike

숫자와 위치가 모두 같은 경우

```java
if(com[i] == user[j] && i == j)
```

### Ball

숫자는 같지만 위치가 다른 경우

```java
if(com[i] == user[j] && i != j)
```

---

### 오늘 공부하며 이해한 점

<ul>
  <li>배열은 반복문과 함께 사용하는 경우가 매우 많다.</li>
  <li>index를 활용하면 배열 데이터를 효율적으로 관리할 수 있다.</li>
  <li>조건문 순서에 따라 결과가 달라질 수 있다.</li>
  <li>2차원 배열은 행과 열 구조를 가진다.</li>
  <li>배열 문제는 반복문 흐름을 이해하는 것이 중요하다.</li>
  <li>이중 for문은 경우의 수 비교와 2차원 배열에서 자주 사용된다.</li>
  <li>숫자 야구게임은 배열 + 반복문 + 조건문의 대표적인 응용 문제이다.</li>
</ul>
