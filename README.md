<h3>1. 변수(Variable)</h3>
<ul>
  <li>데이터를 저장하기 위한 공간</li>
  <li>int, String 등의 자료형 사용</li>
  <li>값을 저장하고 계산 및 비교에 활용</li>
</ul>
'''java
int x = 10;
String name = "Woojin";
'''

---

<h3>조건문 (if / else if / else)</h3>
<p>특정 조건에 따라 실행 흐름을 제어하는 문법</p>

<h5>학습 내용</h5>
<ul>
  <li>특정 범위 조건 검사</li>
  <li>양수, 음수, 0 판별</li>
  <li>배수 판별</li>
  <li>중첩 if문 사용</li>
</ul>

'''java
if(x > 10 && x < 20)
'''

<h5>활용 문제</h5>
<ul>
  <li>50 이상 / 미만 판별</li>
  <li>3의 배수 판별</li>
  <li>놀이기구 탑승 조건 판별</li>
  <li>자판기 메뉴 선택</li>
</ul>
---
<h3>반복문 (for / while / do while)</h3>
<p>같은 작업을 여러 번 반복 실행하는 문법</p>

<h5>for문 기본 구조</h5>
'''java
for(초기식; 조건시기 증감식)
'''

<h5>학습 내용</h5>
<ul>
  <li>짝수 / 홀수 출력</li>
  <li>큰 수부터 출력</li>
  <li>가로 출력 / 세로 출력</li>
  <li>누적합 계산</li>
  <li>이중 for문 사용</li>
</ul>

---

<h3>누적합 계산</h3>
<p>반복문 안에서 값을 계속 더하는 방식</p>

'''java
total += i;
'''

<h5>학습 내용</h5>
<ul>
  <li>0 ~ 10 미만 정수의 합</li>
  <li>특정 범위 홀수 누적합</li>
  <li>2 또는 3의 배수 제외 후 합 계산</li>
  <li>특정 조건의 수만 누적</li>
</ul>

<h5>핵심 개념</h5>
<p>누적 변수는 반복문 밖에서 선언해야 함</p>

'''java
int total = 0;
'''

---
<h3>Scanner 입력</h3>
<p>사용자로부터 값을 입력받기 위한 기능</p>

'''java
Scanner sc = new Scanner(System.in);
'''

<h5>학습 내용</h5>
<ul>
  <li>숫자 입력</li>
  <li>메뉴 선택 입력</li>
  <li>범위 입력</li>
  <li>자판기 프로그램 입력 처리</li>
</ul>
---
<h3>연산자 활용</h3>
<h5>비교 연산자</h5>
'''java
>
<
>=
<=
==
!=
'''
<h5>논리 연산자</h5>
'''java
&&
||
'''
<h5>나머지 연산자</h5>
'''java
%
'''
<h5>활용 예시</h5>
'''java
if (i % 2 == 0)
'''

<ul>
  <li>짝수 / 홀수 판별</li>
  <li>배수 판별</li>
  <li>잔돈 계산</li>
</ul>

---
<h3>이중 for문 활용</h3>
<p>이중 for문을 사용하여 두 개의 주사위 조합 구현</p>

'''java
for(int i = 1; i <= 6; i++) {
  for(int j = 1; j <= 6; j++) {

  }
}
'''

<h5>학습 내용</h5>
<ul>
  <li>모든 경우의 수 탐색</li>
  <li>조건 만족 시 출력</li>
</ul>

'''java
if (i + j == 6)
'''

---

<h3>별 찍기 문제</h3>
<p>이중 for문을 활용한 패턴 출력</p>

'''text
*
**
***
****
*****
'''

<h5>핵심 개념</h5>
<ul>
  <li>바깥 반복문 : 줄 수</li>
  <li>안쪽 반복문 : 출력 개수</li>
</ul>
---

<h3>종합문제 - 자판기 프로그램</h3>
<h5>구현 기능</h5>
<ul>
  <li>현재 보유 금액 입력</li>
  <li>메뉴 출력</li>
  <li>메뉴 선택</li>
  <li>구매 개수 입력</li>
  <li>총 구매 금액 계산</li>
  <li>잔돈 계산</li>
  <li>회폐 단위별 출력</li>
</ul>

<h5>사용 개념</h5>
<ul>
  <li>Scanner</li>
  <li>조건문</li>
  <li>반복문</li>
  <li>누적 계산</li>
  <li>나머지 연산자(%)</li>
</ul>

<h5>잔돈 계산 핵심 로직</h5>
'''java
temp %= 10000;
'''

<h5>학습 내용</h5>
<ul>
  <li>화폐 단위 계산</li>
  <li>필요 없는 잔돈 단위 숨기기</li>
  <li>if문을 활용한 조건 출력</li>
</ul>
---

<h3>이해한 점</h3>
<ul>
  <li>반복문은 출력뿐 아니라 누적 계산에도 자주 사용된다.</li>
  <li>조건문과 반복문은 함께 사용하는 경우가 많다.</li>
  <li>이중 for문은 경우의 수 문제에서 매우 유용하다.</li>
  <li>Scanner를 활용하면 사용자 입력 기반 프로그램을 만들 수 있다.</li>
  <li>'%' 연산자를 활용하면 배수 판별 및 잔돈 계산이 가능하다.</li>
  <li>변수는 선언 위치에 따라 동작 결과가 달라질 수 있다.</li>
</ul>
